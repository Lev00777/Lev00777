### What I do

I build **production-ready AI applications** — not demos. I take a real problem, design the AI architecture around it, integrate the right speech and language models, harden the system for security and privacy, and ship it to real users.

My path into AI is unusual: I came from **18 years on industrial floors**, not from a university lab. My flagship product, *Quiet Translator*, exists because I lived the problem it solves — multilingual communication breaking down in high-noise factories. That's the engineering mindset I bring: **find a real problem → design an AI solution → ship a working product.**

---

### Flagship Product — Quiet Translator

> Real-time, multilingual AI voice translation built for high-noise industrial environments.
> **Live on Google Play & the App Store.**

| | |
|---|---|
| **What it does** | Real-time speech-to-text + neural translation that cuts through machinery noise where standard apps fail |
| **AI pipeline** | Multi-provider system: Deepgram · AssemblyAI · DeepL, with 3 adaptive noise modes |
| **Architecture** | Streaming audio over WebSockets → serverless AWS AI proxy (Lambda + API Gateway) |
| **Privacy** | Zero data retention — audio streamed and discarded instantly, GDPR/CCPA compliant by design |
| **Built by** | One engineer, full lifecycle: AI architecture → UX → security → store deployment |
